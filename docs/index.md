# Lyra Platform Documentation

Welcome to the official Lyra Platform documentation.

## About Lyra

Lyra is a comprehensive multi-tenant platform for managing AI systems, vector stores, and intelligent workflows. Built with modern cloud-native technologies, Lyra provides enterprise-grade features for deploying and managing AI infrastructure.

## Quick Links

- **[Installation & Updates](installation/index.md)** - Get started with deploying Lyra Platform
- **[Support](https://github.com/amreinch/lyra)** - GitHub repository and issue tracker

## Key Features

- **Multi-Tenant Architecture** - Complete isolation between tenants with Kubernetes namespaces
- **AI System Management** - Deploy and manage Ollama, vLLM, and other AI servers
- **Vector Stores** - Integrated vector database management for RAG applications
- **Role-Based Access Control** - Granular permission system with LDAP integration
- **Kubernetes Native** - Built for cloud-native deployments with auto-scaling
- **High Availability** - Production-ready with health checks and monitoring
